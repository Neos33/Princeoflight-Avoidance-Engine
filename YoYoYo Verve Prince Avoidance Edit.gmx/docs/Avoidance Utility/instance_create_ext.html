instance_create_ext

Syntax: 

instance_create_ext(x, y, obj, speed, direction, scale, tag);
Argument name Type Description
x float The X position to spawn the object at.
y float The Y position to spawn the object at.
obj objectID The obejct to spawn.
speed float The initial speed to spawn the object with.
direction float The initial direction the instance will be moving in.
scale float The initial scale of the object. This affects both the X and the Y scale.
tag string The tag to intialize the instance with.

Description:

Creates an instance of an object initialized with speed, direction, scale and tag.
