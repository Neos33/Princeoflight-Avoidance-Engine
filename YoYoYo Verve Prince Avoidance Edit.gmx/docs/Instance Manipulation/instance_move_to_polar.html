instance_move_to_polar

Syntax: 

instance_move_to_polar(targetId, pivotX, pivotY, targetRadius, targetAngle, duration, [easingFunc]);
Argument name Type Description
targetId instanceID The instance to move.
pivotX float The X coordinate of the point to move the instance around.
pivotY float The Y coordinate of the point to move the instance around.
targetRadius float The distance from the pivot point to move the instance to.
targetAngle float The angle to move the instance to, with 0 being the positive horizontal axis (the 3 o'clock position)
duration integer The duration of the change, in frames.
easingFunc easingFuncID The easing function to use. The default is tween_linear.

Description:

Move an instance or all instances of an object to a given position, in polar coordinates around a specified point.

New instance commands override existing ones.
